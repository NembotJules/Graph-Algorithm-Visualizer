# Graph-Algorithm-Visualizer
A graph algorithm visualizer made in Js
# Algorithm Implemented

-Dijkstra <br>
-Kruskal

# Feature 
-Create a new Graph <br>
-Add a Vertex     <br>
-Add Directed Edges  <br>
-Add Undirected Edges <br>
-Move Vertex
